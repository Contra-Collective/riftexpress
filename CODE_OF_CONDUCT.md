# Code of Conduct

This project follows the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

In short: be respectful, be constructive, focus on the work, assume good faith, and don't make people feel unwelcome on the basis of who they are.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers via [SECURITY.md](SECURITY.md)'s contact channel (the same private inbox used for security reports). All reports are reviewed and investigated and will result in a response appropriate to the circumstances. Confidentiality is maintained.

## Scope

Applies to all project spaces — issues, pull requests, code review comments, discussion threads, the wiki, and any RiftExpress-affiliated chat or event — and to public spaces when an individual is representing the project.

## Enforcement

Maintainers who violate this Code of Conduct will be held to the same standard as any contributor. Sustained violations may result in temporary or permanent removal from the project.

For the full enforcement guidelines, see the canonical Contributor Covenant text linked above.
